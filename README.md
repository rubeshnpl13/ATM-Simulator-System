# ATM-Simulator-System
This is a ATM simulator system that contain the feature of banking system too. It is developed on Java using Java Swing and AWT along with MySql as database

## Application 
This system can be used in various banking applications including creating new account, creating account number and pin, 
performing ATM transaction activities.
